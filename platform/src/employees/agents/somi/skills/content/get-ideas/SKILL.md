# get-ideas

Brainstorm content ideas.

## When to Use
User asks for content ideas, inspiration, or what to post about.

## Process
1. Review brand context and past content if available
2. Consider current trends and seasonality
3. Generate diverse ideas:
   - Educational content
   - Behind-the-scenes
   - Engagement posts (polls, questions)
   - Product highlights
   - Storytelling
4. Suggest format for each (post, carousel, reel, thread)

## Output
Return 5-7 ideas with title, description, suggested format, and hook.
