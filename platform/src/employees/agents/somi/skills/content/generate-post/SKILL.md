# generate-post

Generate platform-optimized social media posts.

## When to Use
User asks to create/write/generate a post for any platform.

## Process
1. Check brand profile if available
2. Apply platform rules:
   - **Instagram:** Visual-first, hashtags welcome, 2200 char limit
   - **LinkedIn:** Professional, longer OK, minimal hashtags
   - **Twitter/X:** Punchy, 280 char limit
   - **Facebook:** Conversational, questions work
   - **TikTok:** Trendy, casual, hook-first
3. Generate copy with appropriate tone
4. Add hashtags if platform supports them
5. Show preview before any action

## Output
Return the post content, character count, suggested hashtags, and platform.
