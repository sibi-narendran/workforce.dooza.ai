# show-scheduler

Content calendar view.

## When to Use
- User wants to schedule content
- User asks to see content calendar
- Planning posts for the week/month

## What to Show
- Calendar or list view
- Existing scheduled posts
- Optimal posting times highlighted
- Drag-and-drop scheduling (if supported)

## Views
- Week view (default)
- Month view
- List view
