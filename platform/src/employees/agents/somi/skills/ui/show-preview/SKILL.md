# show-preview

Preview before posting.

## When to Use
- Before publishing anything (automatic)
- User asks to preview or see how it looks
- After generating content

## What to Show
- Platform-specific preview (how it will actually look)
- Character count + warnings
- Media preview if included
- Action buttons: Edit, Schedule, Publish

## Notes
- Mobile-first preview (how most users see content)
- Show truncation points
- Highlight hashtags and mentions
