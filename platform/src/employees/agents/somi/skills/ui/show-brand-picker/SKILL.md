# show-brand-picker

Select brand assets.

## When to Use
- Creating content that needs brand elements
- User wants to pick a logo or color
- Changing brand settings

## What to Show
- Logo variants (full, icon, light, dark)
- Color palette with hex codes
- Font options
- Templates if available

## Actions
- Select asset
- Upload new asset
- Copy color code
