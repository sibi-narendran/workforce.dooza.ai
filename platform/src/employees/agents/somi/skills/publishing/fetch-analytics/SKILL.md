# fetch-analytics

Get performance metrics.

## When to Use
User asks about performance, analytics, metrics, how posts are doing.

## Metrics
- **Reach:** How many saw it
- **Engagement:** Likes, comments, shares, saves
- **Engagement rate:** Engagement / reach
- **Followers gained:** Growth from content

## Timeframes
- Today, this week, this month
- Specific post
- Custom date range

## Notes
- Some metrics delayed 24-48h
- Cross-platform aggregation available
- Identify top performers and patterns
