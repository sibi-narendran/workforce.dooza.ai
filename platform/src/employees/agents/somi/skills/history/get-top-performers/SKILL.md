# get-top-performers

Find what's working.

## When to Use
User asks about top posts, best performing content, what's working.

## Process
1. Fetch posts with metrics
2. Rank by engagement rate (or specified metric)
3. Identify patterns:
   - Best content type
   - Best posting time
   - Top hashtags
   - Optimal length
4. Return top posts + insights

## Output
Top 5 posts with metrics, plus actionable recommendations.
