# meta-tag-generation

Generate SEO-optimized meta tags for web pages.

## When to Use

User asks for title tags, meta descriptions, OG tags, or structured data suggestions.

## Process

1. **Understand the page** ‚Äî what is it about? what's the target keyword?
2. **Check brand profile** for brand name (used in title tag suffix)
3. **Generate multiple options** ‚Äî always provide 2-3 variants
4. **Validate constraints** ‚Äî character limits, keyword placement

## Tag Specifications

### Title Tag
- **Length:** 50-60 characters (Google truncates at ~60)
- **Format:** [Primary Keyword] ‚Äî [Modifier] | [Brand Name]
- **Rules:** Keyword near the front, compelling, unique per page

### Meta Description
- **Length:** 150-160 characters (Google truncates at ~160)
- **Format:** [Action-oriented sentence with keyword]. [Value proposition or CTA].
- **Rules:** Include primary keyword naturally, include a call-to-action, unique per page

### Open Graph Tags
- **og:title** ‚Äî Can differ from title tag (more social-friendly)
- **og:description** ‚Äî Optimized for social sharing (more conversational)
- **og:type** ‚Äî article, website, product, etc.

## Output Format

### Option 1
**Title:** [title tag] ([X] chars)
**Meta Description:** [description] ([X] chars)
**OG Title:** [og:title]
**OG Description:** [og:description]

### Option 2
...

### Recommendation
[Which option and why]

## Tips

- Front-load keywords in title tags
- Use power words in meta descriptions (free, guide, ultimate, proven)
- Include numbers when relevant (10 Tips, 2026 Guide)
- Match search intent in the description
- Don't duplicate tags across pages
